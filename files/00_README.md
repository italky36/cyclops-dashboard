# Задания для ИИ-агента: Раздел «Сделки»

## Порядок выполнения

Задания выполняются **последовательно**, каждое следующее зависит от предыдущего:

1. **Задание 1** — Типы TypeScript и Zod-валидаторы
2. **Задание 2** — API Routes (серверная часть)
3. **Задание 3** — Хуки и утилиты
4. **Задание 4** — UI: Список сделок
5. **Задание 5** — UI: Детальная страница сделки
6. **Задание 6** — UI: Форма создания/редактирования сделки

## Контекст проекта

Перед каждым заданием агенту нужно дать общий контекст:

```
Проект: Cyclops Dashboard
Стек: Next.js 14 (App Router), TypeScript, SQLite, axios, zod, Zustand
Интеграция: Cyclops API (Точка Банк) через JSON-RPC с RSA-подписью
Существующие разделы-примеры: payments, beneficiaries, virtual-accounts
```

- Описание задачи

- Ожидаемый результат (что создать)
